# Be the Hero 
### Seja um Héroi 

Be the hero é uma aplicação voltada para ajudar a comunidade em geral com os mais diversos problemas que a mesma tenha, o
projeto foi desenvolvindo junto com a SemanaOmnistack das RocketSeat.

### Ferramentas
